# Deneme Sınavı Analizi

Fiziksel deneme sınavı için öğrenci girişi, kitapçık görüntüleme ve analiz sayfası (Next.js + Tailwind CSS).

## Renkler

- **Primary:** `#b086fb`
- **Dark:** `#151c37`

## Çalıştırma

```bash
npm install
npm run dev
```

Tarayıcıda [http://localhost:3000](http://localhost:3000) adresine gidin.

## Sayfalar

- **/** — Öğrenci giriş sayfası (e-posta/öğrenci no + şifre)
- **/analiz** — Giriş sonrası: Kitapçık sekmesi + Analiz sekmesi (özet kartlar, ders bazlı alan, grafik alanı). Üstte çıkış butonu.

İçerik (API, veri, PDF görüntüleyici vb.) sonra doldurulacak; şu an sadece tasarım yer alıyor.
